💡 還是不行嗎？常見原因檢查清單
  1. 本機伺服器（Docker backend）沒啟動 → 終端機跑 docker compose up -d
  2. Backend URL 拼錯（例 port 寫錯）→ 點 ⚙️ 齒輪檢查「進階設定 → Backend URL」
  3. 第一次連 LAN backend 沒授權 Chrome 跨來源 → 關掉 popup 重開，當權限對話框跳出時按「允許」
  4. Backend 剛啟動還在 migration（首次啟動會慢）→ 等 30 秒再按「重試」
  5. 防火牆 / VPN 擋住 localhost → 暫時關掉測試

瀏覽你的健康資料
健保存摺資料散在各頁面難瀏覽？「醫析 MediPrisma」把你的 就醫、用藥、檢驗紀錄整理成：
  • 📈 檢驗趨勢 — 膽固醇 / 血糖 / 肝指數變化一目了然
  • 💊 用藥時間軸 — 自己吃過哪些藥、多久
  • 🏥 就醫紀錄 — 哪天去了哪家醫院、看了哪科
  • 🌐 中英對照 — 不熟的英文藥名病名都有中文
  • 🤖 AI 問答 — 看不懂的數值或藥物可隨時詢問
使用方法：點下方按鈕 → 新分頁會打開「醫析 MediPrisma」空白頁面 → 把剛下載的 JSON 檔拖到頁面上、或點頁面上的「匯入資料」 按鈕選取，就會看到你的健康資料。
